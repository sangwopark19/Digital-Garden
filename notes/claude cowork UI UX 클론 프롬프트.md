## 웹사이트 UI/UX 클론 작업

https://wecord.replit.app chrome 확장프로그램을 이용해 모든 경로 페이지의 스크린샷을 찍고 HTML 파일을 분석해서, 이 웹사이트를 최대한 정밀하게 복제하는 UI/UX 마크다운 문서를 만들어줘.

### 작업 순서

1. 스크린샷을 분석해서 페이지의 시각적 구조를 파악해줘
    - 섹션 구분 (Hero, Features, Footer 등)
    - 색상 팔레트 추출 (primary, secondary, background, text 등)
    - 폰트 스타일과 사이즈 체계
    - 간격(spacing)과 레이아웃 패턴
2. HTML 소스에서 추출할 것들
    - 실제 텍스트 콘텐츠 (카피 문구)
    - 이미지 URL들
    - 링크 구조
    - 인터랙션 패턴 (hover, animation 등)
3. 추출한 정보를 기반으로 마크다운문서 생성리
    - 원본과 동일한 애니메이션/트랜지션 구현
    - 모든 파일을 /outputs에 저장

### 중요 규칙

- 원본 스크린샷과 픽셀 단위로 최대한 일치시켜줘
- placeholder 이미지 대신 원본 이미지 URL을 그대로 사용
- "대충 비슷하게"가 아니라 정확한 간격, 색상, 폰트 사이즈를 맞춰줘
- 컴포넌트 네이밍은 원본 사이트의 구조를 반영해줘

**4. `udamon-eas-preview-origin-main`**  
파일: [SKILL.md](/Users/sangwopark19/.codex/skills/udamon-eas-preview-origin-main/SKILL.md:1)

UDAMON에서 “최신 GitHub main 기준 EAS build/link 확인” 전용입니다.  
예: “최신 main 기준 Android/iOS EAS 빌드 링크 줘”, “이 링크가 GitHub 최신 버전 맞아?”

핵심은 현재 checkout을 믿지 않고 `origin/main`을 먼저 fetch해서 SHA를 확인하는 겁니다.

- `git fetch --prune origin`
- `origin/main` SHA 확인
- 필요하면 detached worktree 생성
- `expo-doctor`
- `eas build:list --git-commit-hash <sha>`
- `build:view --json`으로 artifact URL 확인

즉, 현재 feature branch를 더럽히지 않고 “진짜 GitHub 최신 main 빌드인지”를 검증하게 합니다.
